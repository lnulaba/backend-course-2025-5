# Лабораторна робота №5: Примітивний кешуючий проксі-сервер

## Опис
Цей проект реалізує примітивний кешуючий проксі-сервер на базі модуля `node:http`. Сервер кешує картинки з сайту https://http.cat та зберігає їх локально для швидшого доступу при повторних запитах.

## Функціонал
- **GET** - отримання картинки з кешу або з http.cat
- **PUT** - збереження картинки в кеш
- **DELETE** - видалення картинки з кешу
- Автоматичне створення директорії для кешу
- Підтримка HTTP статусних кодів

## Використання

### Запуск сервера
```bash
node server.js --host localhost --port 3000 --cache ./cache
```

### Параметри командного рядка
- `-h, --host` - адреса сервера (обов'язковий)
- `-p, --port` - порт сервера (обов'язковий)  
- `-c, --cache` - шлях до директорії кешу (обов'язковий)

### Приклади запитів
```bash
# Отримання картинки для HTTP коду 200
curl -X GET http://localhost:3000/200 --output image_200.jpg

# Збереження картинки в кеш
curl -X PUT http://localhost:3000/201 --data-binary @image.jpg

# Видалення картинки з кешу
curl -X DELETE http://localhost:3000/200
```

## Технології
- Node.js
- Commander.js - для роботи з аргументами командного рядка
- Superagent - для HTTP запитів
- Nodemon - для автоматичного перезапуску під час розробки
